DAY3 10/11/65
วันนี้ผมได้ตื่นมาตอน 8 โมงเพื่อเตรียมตัวไปเรียนที่มหาลัยที่มีสอนตอน 8:30 ซึ่งจะเรียนเกี่ยวกับภาษา JAVA เลิกเรียนเวลา 12:30 เมื่อเรียนเสร็จแล้วผมก็ไปหาอะไรกินกับเพื่อนๆ ซึ่งตอนเย็นวันนี้ผมมีค่าย Hello World ผมเลยกลับไปที่หอพักของผมเพื่อนอนพักผ่อนก่อนที่จะไปค่าย Hello World โดยผมทำหน้าที่เป็น Web Design ซึ่งเป็นครั้งแรกที่ผมได้เรียนรู้เกี่ยวกับ Figma ซึ่งเป็นการออกแบบหน้าเว็บ ผมก็ได้ลองทำ Workshop ที่พี่ๆตั้งโจทย์มาให้ เป็นการเรียนที่สนุกมากครับ ซึ่งพี่ๆก็ได้ปล่อยพวกผมเป็นเวลา 19:30 เมื่อพวกผมเลิกกิจกรรมนี้ก็ได้ออกไปหาไรกินกับเพื่อนๆต่อซึ่งก็ได้ไปกินซูชิหลังมหาลัย อร่อยมาก! แล้วก็ได้เตรียมตัวแยกย้ายกลับหอพัก วันนี้เป็นวันที่สนุกมากเลยครับ ผมก็ได้อาบน้ำ แล้วก็พักผ่อน เข้านอนครับ
